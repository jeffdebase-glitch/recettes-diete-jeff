# On mange quoi aujourd’hui ?

Le carnet diète de Jeff.

Site statique mobile-first pour retrouver les recettes Creami, salées et desserts compatibles avec une sèche.

## Mise en ligne

1. Activer GitHub Pages sur la branche `main`, dossier `/root`.
2. Ouvrir le site publié.
3. Plus tard, renseigner `config.js` avec une URL Google Sheets CSV si besoin.
